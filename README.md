Ir. M Hafiz | Professional Landing Page

This repository contains the source code for the personal landing page of Ir. M Hafiz, a Professional Engineer and Data Analyst. The site is designed to be a modern, high-performance "Link-in-bio" style profile that highlights professional credentials and social connectivity.

🚀 Live Preview

The site is hosted via GitHub Pages at: azmimh.github.io

✨ Key Features

Professional Identity: Specifically tailored for an engineering and data analytics professional.

Modern UI/UX: Features a sleek "Glassmorphism" aesthetic with a responsive dark-mode palette.

Categorized Links: Separated into "Professional" (LinkedIn, Tableau, Email) and "Social Media" (FB, X, Instagram) for better user navigation.

Fully Responsive: Optimized for seamless viewing on mobile, tablet, and desktop devices.

Performance Optimized: Built using Tailwind CSS and hosted directly on GitHub Pages for lightning-fast load times.

🛠️ Technology Stack

HTML5: Semantic structure.

Tailwind CSS: Utility-first styling for modern design tokens and responsiveness.

FontAwesome: High-quality vector icons for social branding.

Google Fonts: Utilizing the 'Inter' typeface for maximum legibility.

JavaScript: Minor script for dynamic copyright year updates.
